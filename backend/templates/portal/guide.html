Security Guide

How to spot suspicious email, how to report it, and what Sentinel's verdicts mean.

Common warning signs

Urgency or pressure language
"verify now", "your account will be suspended", "act immediately"
Requests for sensitive information
Passwords, card numbers, or verification codes — legitimate services rarely ask by email
Sender / brand mismatch
Display name says "PayPal" but the domain is something else entirely
Links that don't go where they claim
Displayed text shows one domain, the real destination is another
Generic greetings
"Dear Customer" instead of your name

How to get the original .eml file

The original file preserves headers, authentication results, and real link destinations — pasted text cannot.

Gmail
Open the message → ⋮ (more) → "Show original" → "Download original"
Outlook
Drag the message from the list onto your desktop, or File → Save As

What Sentinel's verdicts mean

LegitimateLow estimated risk — no action needed
Needs ReviewUncertain — an administrator will investigate
PhishingHigh estimated risk — quarantined where possible

Phishing probability is the model's estimated likelihood a message is phishing. Prediction confidence is how sure the model is of whichever label it picked — a message can be 95% confidently legitimate, which is different from being "5% phishing".

Why human review matters

Sentinel's classifier achieves strong accuracy on its held-out test set, but no automated system is perfect. "Needs Review" exists so uncertain cases go to a person instead of being silently guessed one way or the other — and every automated verdict can be corrected by an administrator.

Still unsure about a message? Report it — a false alarm costs a minute of an administrator's time; a missed phishing email can cost far more.
