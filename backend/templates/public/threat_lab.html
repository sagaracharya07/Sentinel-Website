Understand the indicators Sentinel looks for

Educational reference — not live detection. See Live Demo for an interactive walkthrough, or the Administrator Console's Incident Detail page for real findings.

Phishing

A message designed to trick the recipient into revealing credentials, making a payment, or installing malware — usually by impersonating a trusted sender.

Business email compromise

An attacker impersonates an executive, vendor, or colleague to request a wire transfer, gift cards, or sensitive data — often with no malicious link or attachment at all.

Credential harvesting

A fake login page collects a victim's username and password, usually reached via a link that looks legitimate but resolves to an attacker-controlled domain.

Display-name impersonation

The visible sender name says "PayPal Security" or "IT Support" while the actual email address belongs to an unrelated or lookalike domain.

Lookalike domains

Domains that visually resemble a trusted one — substituting characters like "0" for "o", or adding a hyphen — to slip past a quick glance.

Reply-To deception

The From address looks legitimate, but the Reply-To header quietly redirects any response to an attacker-controlled mailbox.

Suspicious links

Displayed link text that doesn't match its real destination, URL shorteners hiding the true target, or raw IP-address destinations.

Suspicious attachments

Executable, script, or macro-enabled file types, double extensions (e.g. invoice.pdf.exe), or archives concealing their real content.

SPF, DKIM & DMARC

Email authentication standards that let a receiving server verify a message really came from where it claims. Sentinel reads the provider's authentication results — it does not perform independent DNS validation.

False positives

A legitimate message incorrectly flagged as risky. Sentinel's "Needs Review" state and human review process exist specifically to catch and correct these.

Human review

No automated classifier is perfect. An administrator can confirm, release, or override any verdict — and every override is recorded.
