Protect · Investigate · Respond

One console for the full lifecycle

From the moment mail arrives in a connected Gmail mailbox to the moment an administrator makes a final call.

Gmail monitoring

Connect a mailbox once with Google OAuth. Sentinel detects changes through the Gmail API and analyses every new message automatically.

Three-state detection

Every message is classified Legitimate, Needs Review, or Phishing — with a phishing probability and a prediction confidence, not a single opaque score.

Structured investigation

Sender identity, Reply-To and Return-Path, authentication results, link destinations, and attachment metadata — as explainable findings, not a black box.

Quarantine & release

Phishing is quarantined in Gmail using label operations — never deleted. Release returns a message to the inbox once an administrator confirms it's safe.

Employee reporting

Anyone can upload a suspicious .eml for the same full analysis, then track its status through to an administrator's final verdict.

Full audit trail

Every connection, detection action, and administrator decision is recorded — searchable, and never silently overwritten.
