Changelog

What's actually shipped

2026
Feature

Full site revamp + self-serve accounts

Sign-up, email verification, and password reset, backed by real outbound email. Ten-plus new pages (Features, Pricing, Security, Integrations, and more) sharing one consistent template instead of hand-duplicated pages.

2026
Infra

S3-compatible object storage for model artifacts

Retrained models are now handed off between the background worker and the web service through object storage, so a new model reaches production traffic correctly even when they run as fully separate services.

2026
Infra

Postgres, Celery, and Docker

Migrated off single-file SQLite and in-process background threads onto Postgres and a Celery/Redis job queue, with Alembic-managed migrations, a full Docker Compose setup for local development, and a Render deployment configuration.

2026
Security

Security hardening pass

Closed an insecure default session secret, added rate limiting on authentication and retraining endpoints, tightened security headers (CSP/HSTS), and gated demo-account seeding out of production entirely.

2026
Model

v1 model trained

Initial Random Forest classifier trained on ~37,700 labelled emails across four public corpora — 96.1% accuracy, 94.9% precision, 96.8% recall on a held-out test split.
